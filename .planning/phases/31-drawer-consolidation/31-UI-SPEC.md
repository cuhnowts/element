## Interaction Contract

### Tab Bar Behavior (DRAW-01)

| Interaction | Result |
|-------------|--------|
| Click tab when drawer is collapsed | Drawer expands to `drawerHeight` (last drag position from `useWorkspaceStore`); clicked tab becomes active |
| Click active tab when drawer is expanded | Drawer collapses (panel height 0%) |
| Click inactive tab when drawer is expanded | Active tab switches; drawer stays expanded |
| Drag resize handle | Standard `react-resizable-panels` behavior; stores new `drawerHeight` |

### Tab Order (D-06)

Position 1: **Element AI** | Position 2: Terminal | Position 3: Logs | Position 4: History

The "Runs" tab renders conditionally only when a workflow is selected (existing behavior, unchanged).

### Element AI Pane Internal Layout (DRAW-02, D-04, D-05)

```
+---------------------------------------------------------------+
| [status dot] Agent [status label]    [Activity] [Terminal]     |
+---------------------------------------------------------------+
|                                                                |
|  (AgentActivityTab or AgentTerminalTab fills remaining height) |
|                                                                |
+---------------------------------------------------------------+
```

- Sub-tab header: 40px height (`h-10`), bottom border (`border-b border-border`), horizontal padding 16px (`px-4`), vertical padding 8px (`py-2`)
- Left side: status dot (8px circle, `size-2 rounded-full`), "Agent" label (14px semibold), status text (12px muted)
- Right side: Activity and Terminal toggle buttons using existing `tabClass` pattern (`text-xs font-semibold tracking-wide uppercase`)
- No close chevron (the drawer tab bar handles collapse)
- "Restart Agent" button appears when status is "stopped" (existing behavior, carried forward)

### Pending Approval Badge (relocated from AgentToggleButton)

When `pendingApprovalCount > 0`, show a destructive badge on the "Element AI" tab label in the drawer tab bar:
- Badge: `size-4 p-0 text-[10px] justify-center absolute -top-1 -right-1`
- Position the tab label wrapper as `relative` to anchor the badge
- Display count (cap at "9+" for > 9)
- This replaces the badge that was on the deleted `AgentToggleButton`

### Agent Lifecycle (D-10, D-11, D-12)

- `startAgent()` called at AppLayout mount (not tied to Element AI tab visibility)
- `useAgentQueue()` remains at AppLayout level (already there)
- Switching away from Element AI tab does NOT stop or pause the agent
- Agent state changes (status dot color) reflect in the sub-tab header even when the tab is not visible (mount-all/show-one pattern)

### Drawer Collapse/Expand Animation

CSS transition on the `ResizablePanel` collapse/expand -- `react-resizable-panels` handles this natively. No additional animation library needed. If the library default is instant, add `transition: flex-basis 150ms ease` to the drawer panel.
