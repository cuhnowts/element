In `src/components/layout/AppLayout.tsx`:

1. Add `useEffect` and `useRef`/callback ref imports. Import `usePanelRef` from `react-resizable-panels`:
   ```
   import { useEffect } from "react";
   import { usePanelRef } from "react-resizable-panels";
   ```
   (Keep the existing `useState` import.)

2. Inside the `AppLayout` component, create a panel ref using the convenience hook:
   ```
   const drawerPanelRef = usePanelRef();
   ```

3. Add a `useEffect` that imperatively collapses or expands the drawer panel when `drawerOpen` changes:
   ```
   useEffect(() => {
     const panel = drawerPanelRef.current;
     if (!panel) return;
     if (drawerOpen) {
       panel.expand();
     } else {
       panel.collapse();
     }
   }, [drawerOpen, drawerPanelRef]);
   ```

4. Pass `panelRef` to the drawer's `ResizablePanel` (the second one, around line 87-94). Change:
   ```
   <ResizablePanel
     defaultSize={`${drawerOpen ? drawerHeight : 0}%`}
     minSize="0%"
     maxSize="60%"
     collapsible
   >
   ```
   to:
   ```
   <ResizablePanel
     defaultSize={`${drawerOpen ? drawerHeight : 0}%`}
     minSize="0%"
     maxSize="60%"
     collapsible
     panelRef={drawerPanelRef}
   >
   ```

This wires the imperative API: when useKeyboardShortcuts calls `toggleDrawer()` (which flips `drawerOpen` in the workspace store), the `useEffect` fires and calls `panel.collapse()` or `panel.expand()`, actually resizing the panel. Previously, only `defaultSize` changed, which has no effect after initial render.
