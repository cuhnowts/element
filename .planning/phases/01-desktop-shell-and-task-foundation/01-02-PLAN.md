    2. Create src-tauri/src/commands/project_commands.rs:
       Each command takes `app: AppHandle` and `state: State<'_, std::sync::Mutex<Database>>`.
       Lock the mutex briefly: `let db = state.lock().map_err(|e| e.to_string())?;`

       Commands (all return `Result<T, String>` where errors are `.map_err(|e| e.to_string())`):

       ```rust
       #[tauri::command]
       pub async fn create_project(
           app: AppHandle,
           state: State<'_, std::sync::Mutex<Database>>,
           name: String,
           description: Option<String>,
       ) -> Result<Project, String> {
           let db = state.lock().map_err(|e| e.to_string())?;
           let input = CreateProjectInput { name, description };
           let project = db.create_project(input).map_err(|e| e.to_string())?;
           app.emit("project-created", &project).map_err(|e| e.to_string())?;
           Ok(project)
       }
       ```

    3. Create src-tauri/src/commands/task_commands.rs:
       Same pattern with Mutex<Database> state.

       Commands:
       - `create_task(app, state, project_id, title, description?, context?, priority?, external_path?)` -> Task, emits "task-created"
       - `list_tasks(state, project_id)` -> Vec<Task>
       - `get_task(state, task_id)` -> Task (include tags via get_tags_for_task)
       - `update_task(app, state, task_id, title?, description?, context?, priority?, external_path?)` -> Task, emits "task-updated"
       - `update_task_status(app, state, task_id, status: String)` -> Task, emits "task-updated"
         Parse status string to TaskStatus enum. Valid values: "pending", "in-progress", "complete", "blocked".
       - `delete_task(app, state, task_id)` -> (), emits "task-deleted"
       - `add_tag_to_task(app, state, task_id, tag_name: String)` -> Tag
         Create tag if it doesn't exist (get_or_create pattern), then add to task. Emit "task-updated".
       - `remove_tag_from_task(app, state, task_id, tag_id: String)` -> (), emits "task-updated"
       - `list_tags(state)` -> Vec<Tag> (all tags in system)

       For `get_task`, return an extended struct or include tags in the response:
       ```rust
       #[derive(Serialize)]
       #[serde(rename_all = "camelCase")]
       pub struct TaskWithTags {
           #[serde(flatten)]
           pub task: Task,
           pub tags: Vec<Tag>,
       }
       ```
