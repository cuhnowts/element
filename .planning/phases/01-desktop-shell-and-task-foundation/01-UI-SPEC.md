## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, inline padding within badges |
| sm | 8px | Compact element spacing, task row internal padding |
| md | 16px | Default element spacing, sidebar section padding |
| lg | 24px | Section padding, task detail field spacing |
| xl | 32px | Layout gaps between sidebar and main area |
| 2xl | 48px | Empty state vertical spacing |
| 3xl | 64px | Not used in this phase |

Exceptions: Sidebar drag handle hit area is 12px wide (not on the 4px scale) to match shadcn ResizablePanelGroup defaults.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 500 (medium) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

Notes:
- Body at 14px matches Linear/Discord density for desktop apps (not 16px web default).
- Label is used for sidebar section headers, badge text, status text, and metadata.
- Heading is used for task detail title and project names.
- Display is used only for the empty state onboarding headline.

## Color

shadcn default theme uses OKLCH. All values below are the CSS custom property names from shadcn's theming system. The app ships with both light and dark modes, defaulting to OS preference via Tailwind v4's `prefers-color-scheme`.

| Role | Light Value | Dark Value | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `--background` oklch(1 0 0) | `--background` oklch(0.145 0 0) | App background, main content area |
| Secondary (30%) | `--card` oklch(0.985 0 0) | `--card` oklch(0.205 0 0) | Sidebar background, task detail card, input fields |
| Accent (10%) | `--primary` oklch(0.488 0.243 264.376) | `--primary` oklch(0.623 0.214 259.815) | See reserved-for list below |
| Destructive | `--destructive` oklch(0.577 0.245 27.325) | `--destructive` oklch(0.577 0.245 27.325) | Delete task, delete project |

Accent color is a blue (hue ~264) chosen to match the productivity-tool aesthetic of Linear, Discord, and VS Code. This overrides shadcn's default grayscale primary.

**Accent reserved for:**
- Primary CTA buttons ("Create Project", "Create Task")
- Selected/active task row highlight in sidebar
- Selected/active project indicator in sidebar
- Status badge: "In Progress" only (other statuses use neutral or semantic colors)
- Focus ring on interactive elements

**Status colors (semantic):**
| Status | Color | Token |
|--------|-------|-------|
| Pending | Muted foreground | `--muted-foreground` |
| In Progress | Accent/primary | `--primary` |
| Complete | Green | oklch(0.627 0.194 149.214) |
| Blocked | Destructive/red | `--destructive` |

**Priority badge colors:**
| Priority | Style |
|----------|-------|
| Urgent | Destructive background, destructive foreground |
| High | Orange outline badge — oklch(0.705 0.213 47.604) border, transparent bg |
| Medium | Muted background, muted foreground (default appearance) |
| Low | No badge displayed (absence communicates low priority) |

## Interaction Patterns

### Sidebar behavior
- Default sidebar width: 280px, min 200px, max 400px
- Sidebar is resizable via drag handle (shadcn ResizablePanelGroup)
- Project list in top section, task list in bottom section
- Clicking a project loads its tasks in the bottom section
- Clicking a task loads its detail in the main area

### Task list rows
- Single line per task: `[status icon] [title] [priority badge]`
- Selected row uses accent background at 10% opacity
- Hover state uses muted background
- Right-click opens context menu (edit, change status, delete)

### Task detail panel
- Field order: Title (editable heading), Status (select), Priority (select), Tags (inline badges), Description (textarea), Context (textarea/markdown)
- Inline editing: fields are directly editable, no separate "edit mode"
- Auto-save on blur or after 1 second debounce of typing

### Keyboard shortcuts
- `Cmd+K`: Open command palette
- `Cmd+N`: New task in current project
- `Cmd+Shift+N`: New project
- `Cmd+Backspace`: Delete selected task (with confirmation dialog)
- `Up/Down arrows`: Navigate task list
- `Enter`: Select/open highlighted task
- `Escape`: Close dialogs, deselect task

### System tray
- App icon in system tray
- Tray menu: "Show Element", "Quit"
- Closing window hides to tray (does not quit)

### Loading states
- Skeleton placeholders for task list and task detail during initial data load
- Button spinner (Lucide Loader2 icon, animated) for save operations

### Dark/light mode
- Follows OS preference on first launch
- Manual toggle available in app header/toolbar
- Persists user preference in SQLite after manual toggle
