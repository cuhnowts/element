Add to `WorkspaceState` interface:
```typescript
projectStates: Record<string, ProjectWorkspaceState>;
getProjectState: (projectId: string) => ProjectWorkspaceState;
setProjectCenterTab: (projectId: string, tab: "detail" | "files") => void;

```

Default values for a new project (returned by `getProjectState` when no entry exists):
```typescript
const DEFAULT_PROJECT_STATE: ProjectWorkspaceState = {
  centerTab: "detail",
  drawerOpen: true,   // match global default
  drawerTab: "logs",  // match global default
};
```

Implementation:
- `getProjectState(projectId)`: return `projectStates[projectId] ?? DEFAULT_PROJECT_STATE`
- `setProjectCenterTab(projectId, tab)`: update `projectStates[projectId].centerTab`
- `setProjectDrawerState(projectId, drawerOpen, drawerTab)`: update drawer fields
- `saveCurrentProjectState(projectId)`: snapshot current global `drawerOpen` and `activeDrawerTab` into `projectStates[projectId]`, also read the current `activeProjectTab` from the store to save as `centerTab`
- `restoreProjectState(projectId)`: read from `projectStates[projectId]` and apply to global `drawerOpen` and `activeDrawerTab`
- `launchTerminalCommand(command, args)`: set `terminalInitialCommand` to `{ command, args }`, increment `terminalSessionKey` by 1, and call `openDrawerToTab("terminal")`. Incrementing the session key changes the TerminalTab's React key, causing unmount (kills old PTY) and remount (starts new PTY with the command).

Initialize in store: `projectStates: {}`, `terminalSessionKey: 0`, `terminalInitialCommand: null`.

**CRITICAL**: Do NOT persist `projectStates`, `terminalSessionKey`, or `terminalInitialCommand` in the `partialize` function. They stay in-memory and reset on restart.

**Implementation approach (simpler — sync via existing global state)**:
```typescript
const prevProjectRef = useRef<string | null>(null);

useEffect(() => {
  if (!selectedProjectId) return;

  // Save outgoing project state
  if (prevProjectRef.current && prevProjectRef.current !== selectedProjectId) {
    saveCurrentProjectState(prevProjectRef.current);
  }

  // Restore incoming project state
  restoreProjectState(selectedProjectId);
  const projectState = getProjectState(selectedProjectId);
  setActiveProjectTab(projectState.centerTab);

  prevProjectRef.current = selectedProjectId;
}, [selectedProjectId]);
```
