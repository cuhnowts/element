# Phase 2.1 Plan 04: Quick Capture Window Summary

**Global hotkey Ctrl+Space opens frameless floating capture window with task form, cross-window event refresh via Tauri multi-window architecture**

## Performance

- **Duration:** 8 min (across two sessions including human verification)
- **Started:** 2026-03-16
- **Completed:** 2026-03-16
- **Tasks:** 3 (2 auto + 1 human-verify checkpoint)
- **Files modified:** 5

## Accomplishments
- System-wide Ctrl+Space global shortcut opens a frameless floating capture window center-screen
- Quick-capture form with title (auto-focused), project selector, priority selector, due date picker, and duration chips
- Enter saves task and dismisses window, Escape dismisses without saving
- Cross-window event emission triggers automatic today view refresh in main window
- Multi-window Tauri routing established via URL query parameter pattern

1. **Task 1: Configure capabilities, routing, global shortcut hook, and QuickCaptureWindow component** - `c2daf77` (feat)
2. **Task 2: Wire useGlobalShortcut into AppLayout and verify cross-window event path** - `8db57c4` (feat)
3. **Task 3: Verify quick-capture end-to-end** - human-verify checkpoint (approved, no commit)

## Files Created/Modified
- `src-tauri/capabilities/default.json` - Added capture window permissions and webview creation capability
- `src/main.tsx` - Multi-window routing: checks URL param to render QuickCaptureWindow or main App
- `src/hooks/useGlobalShortcut.ts` - Registers Ctrl+Space, creates/toggles capture WebviewWindow
- `src/components/capture/QuickCaptureWindow.tsx` - Frameless capture form with title/project/priority/due/duration fields
- `src/components/layout/AppLayout.tsx` - Wired useGlobalShortcut hook for shortcut registration on mount

## Decisions Made
- Multi-window routing via URL query param (?window=capture) with dynamic imports in main.tsx
- Capture window uses direct invoke() calls instead of Zustand stores to avoid cross-window state sharing
- Frameless window with decorations:false and CSS border-radius instead of transparent:true to avoid macOS issues
