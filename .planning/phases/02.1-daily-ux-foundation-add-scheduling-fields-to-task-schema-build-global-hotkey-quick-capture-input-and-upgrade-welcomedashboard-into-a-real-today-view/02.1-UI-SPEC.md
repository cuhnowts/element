## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline chip padding, status dot margins |
| sm | 8px | Compact element spacing, chip gaps, task row internal padding |
| md | 16px | Default element spacing, section content padding |
| lg | 24px | Section padding, today view section gaps |
| xl | 32px | Quick-capture window internal padding |
| 2xl | 48px | Today view header bottom margin |
| 3xl | 64px | Not used in this phase |

Exceptions: Quick-capture window has 12px border-radius (3 * xs) for a softer floating appearance. Touch targets for duration chips are 32px min-height.

## Interaction Contracts

### Quick-Capture Window

| Property | Value |
|----------|-------|
| Trigger | Ctrl+Space (global hotkey, system-wide via Tauri global-shortcut plugin) |
| Window size | 560px wide, auto-height (max 320px), centered on screen |
| Window style | Frameless, transparent background with card-colored surface, 12px border-radius, 1px border --color-border, drop-shadow-lg |
| Appear animation | Fade in 150ms ease-out + scale from 0.95 to 1.0 |
| Dismiss animation | Fade out 100ms ease-in |
| Auto-focus | Title input receives focus immediately on window appear |
| Submit | Enter key saves task, window dismisses |
| Cancel | Escape key dismisses window without saving |
| Post-submit | Window disappears. Task created in backend. Optional toast in main window confirming creation. |
| Field layout | Title input (full width, top) -> row of: project selector, priority selector, due date button, duration chips (below title) |
| Defaults | Project: currently selected project in main window. Priority: medium. Due date: none. Duration: none. |

### Today View

| Property | Value |
|----------|-------|
| Mount condition | Center panel renders TodayView when no task is selected (replaces WelcomeDashboard) |
| Scroll | Vertical scroll via ScrollArea, full height of center panel |
| Data scope | All tasks across all projects that have scheduledDate = today, OR dueDate <= today, OR status != complete with no date (unscheduled) |
| Refresh | Reactive via Zustand store. Re-renders on task create/update/delete events. |

### Today View Header

| Property | Value |
|----------|-------|
| Greeting | "Good morning" / "Good afternoon" / "Good evening" based on current hour (reuse existing logic from WelcomeDashboard) |
| Greeting size | Display (28px/600) |
| Progress line | Body (14px/400): "{completed} of {total} tasks complete" |
| Progress bar | 100% width, 4px height, border-radius 2px. Track: --color-secondary. Fill: --color-primary. Width = completed/total percentage. |
| Stats line | Label (12px/600): "{overdue count} overdue / {upcoming count} upcoming" — overdue count uses --color-destructive-foreground |
| Bottom spacing | 2xl (48px) margin-bottom before first time group |

### Time Group Sections

| Property | Value |
|----------|-------|
| Groups (in order) | Overdue, Morning, Afternoon, Evening, Unscheduled |
| Time boundaries | Morning: scheduledTime < 12:00. Afternoon: 12:00 <= scheduledTime < 17:00. Evening: scheduledTime >= 17:00. |
| Overdue rule | dueDate < today AND status not "complete" |
| Unscheduled rule | No scheduledDate and no dueDate, status not "complete" |
| Section header | Label (12px/600), uppercase, tracking-wide, text-muted-foreground. Left-aligned. |
| Overdue header | Same as above but text uses --color-destructive-foreground |
| Empty groups | Hidden entirely (do not render header if group has zero tasks) |
| Gap between groups | lg (24px) |
| Gap between task rows | xs (4px) |

### Today Task Row

| Property | Value |
|----------|-------|
| Height | Auto, min 36px, padding-y 8px (sm) |
| Layout | Horizontal flex: StatusDot (left) + title (flex-1, truncate) + project label (right) + due badge (right) |
| Hover | bg-muted transition-colors 150ms |
| Click | Selects task (same as existing task list behavior) |
| Completed state | opacity-40 on entire row, line-through on title text |
| "Next up" state | 3px left border --color-primary at 60% opacity. Applied to first incomplete, non-overdue task in chronological order. |
| Overdue badge | Badge variant="destructive", text "Overdue" — shown only in non-Overdue sections if task is overdue |
| Project label | Label (12px/400), text-muted-foreground, max-width 120px, truncate |
| Due date indicator | Label (12px/600), text-muted-foreground. Format: "Due Mon", "Due Mar 20". Omitted if no due date. |

### Scheduling Badges (Task Detail)

| Property | Value |
|----------|-------|
| Location | Task detail header area, inline row below title |
| Layout | Horizontal flex-wrap, gap sm (8px) |
| Due date badge | Badge variant="outline". Icon: CalendarDays (16px). Text: formatted date. If overdue: variant="destructive". |
| Scheduled badge | Badge variant="outline". Icon: Clock (16px). Text: "Mon 2:00 PM" (date + time). |
| Duration badge | Badge variant="secondary". Icon: Timer (16px). Text: "30m" / "1h" / etc. |
| Recurrence badge | Badge variant="secondary". Icon: Repeat (16px). Text: "Daily" / "Weekdays" / "Weekly" / "Monthly". |
| Empty state | If no scheduling fields set, no badges rendered (no placeholder). |

### Duration Chips

| Property | Value |
|----------|-------|
| Presets | 15m, 30m, 1h, 2h, 4h |
| Custom | "Custom" chip opens inline number input (minutes) |
| Layout | Horizontal flex, gap xs (4px) |
| Chip size | min-width 40px, height 32px, padding-x sm (8px), border-radius 6px |
| Inactive chip | bg --color-secondary, border --color-border, text --color-foreground, Label (12px/600) |
| Active chip | bg --color-primary, text --color-primary-foreground, Label (12px/600) |
| Single select | Only one chip active at a time. Clicking active chip deselects (sets duration to null). |

## Accessibility Notes

| Concern | Treatment |
|---------|-----------|
| Quick-capture window | role="dialog", aria-label="Quick capture task". Focus trap within window. |
| Duration chips | role="radiogroup" on container, role="radio" + aria-checked on each chip |
| Progress bar | role="progressbar", aria-valuenow, aria-valuemin="0", aria-valuemax="100", aria-label="Task completion progress" |
| Overdue badge | aria-label="Overdue" on badge for screen readers |
| Time group sections | Use heading element (h2) for section headers with appropriate aria-level |
| Keyboard nav in today view | Arrow keys to navigate task rows, Enter to select |
| Color contrast | All text meets WCAG AA on dark background. Destructive foreground (oklch 0.637) on background (oklch 0.145) = sufficient contrast. |
