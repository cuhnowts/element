## Interaction Contracts

### Sidebar Project Click (UI-01)
- **Left-click on project name**: Navigate to ProjectDetail in center panel. No menu, no dialog, no delay.
- **Left-click on chevron**: Toggle inline task list expand/collapse. Does NOT navigate. Event uses stopPropagation.
- **Right-click on project name**: Open context menu with Move to Theme, Delete.
- **Keyboard**: Enter key on focused project row triggers navigation (same as left-click).

### Theme Collapse (UI-02)
- **Click on theme header chevron**: Toggle theme section expand/collapse.
- **Persistence**: State saved to useWorkspaceStore immediately on toggle. Survives app restart.
- **Animation**: Use Collapsible component's default data-open/data-closed CSS transitions.

### Task Detail Accordion (UI-03)
- **Click accordion trigger**: Toggle that section open/closed. Multiple sections can be open simultaneously (type="multiple").
- **No auto-close**: Opening one section does NOT close others.
- **Keyboard**: Space/Enter on focused trigger toggles section.

### AI Button State Machine (UI-05, UI-07)
- **No directory**: Button disabled with tooltip. DirectoryLink is the primary CTA on the row.
- **Has directory, no tier**: Button enabled, clicking opens tier selection dialog.
- **Has tier/content**: Button enabled, clicking launches AI in terminal.
- **Executing**: Button shows spinner, still clickable (opens terminal to view progress).

### Drawer Tab Default (UI-06)
- **First open**: Terminal tab is selected by default.
- **Tab persistence**: Per-project tab selection within a session (existing behavior, no change).

## State Derivation Contract

The AI button state machine is the primary new logic in this phase. It MUST be implemented as a pure function for testability.

### getAiButtonState() Input/Output

```
Input:
  directoryPath: string | null
  planningTier: string | null
  hasContent: boolean
  isExecuting: boolean (derived from terminal/launch state)

Output:
  label: "Link Directory" | "Plan Project" | "Check Progress" | "Open AI"
  disabled: boolean
  tooltip: string | undefined
  showSpinner: boolean
```

### State Priority (evaluated top to bottom)

| Priority | Condition | Label | Disabled | Spinner |
|----------|-----------|-------|----------|---------|
| 1 | directoryPath is null | "Link Directory" | true | false |
| 2 | planningTier is null AND hasContent is false | "Plan Project" | false | false |
| 3 | isExecuting is true | "Open AI" | false | true |
| 4 | hasContent is true | "Check Progress" | false | false |
| 5 | fallback | "Open AI" | false | false |
