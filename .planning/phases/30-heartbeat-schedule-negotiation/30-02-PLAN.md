    **run_heartbeat_tick(app_handle: &AppHandle) -> Result<(), String>**:
    1. **Scoped DB lock** -- extract all needed data then drop lock immediately (per Pitfall 1):
       ```rust

           let work_hours = read_work_hours_config(&db)?;
           let today = chrono::Local::now().date_naive();
           let max_due = tasks.iter().map(|t| t.due_date).max().unwrap_or(today);

       }; // lock dropped
       ```

       {
           let db = db_state.lock().map_err(|e| e.to_string())?;
