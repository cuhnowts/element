## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 24px | 600 (semibold) | 1.2 |
| Goal Text | 16px | 400 (regular) | 1.5 |

Label is distinguished from Body by uppercase transforms, muted color (`--color-muted-foreground`), or tighter letter-spacing -- not by font weight.

Goal Text is 16px to provide visual prominence above the 14px body without competing with the 24px project name heading.

Source: existing `body { font-size: 14px; line-height: 1.5 }` in app.css. Heading matches existing project name at `text-2xl font-semibold`.

## Interaction Contract

### Goal Editing (PROJ-02)

| State | Behavior |
|-------|----------|
| Display | Goal text shown with Target icon; edit pencil visible on hover |
| Click pencil | Goal text replaced by focused `Input`; cursor at end of text |
| Click empty card | Enters edit mode with empty input focused |
| Blur / Enter | Saves via `api.updateProject()` with debounced auto-save; returns to display mode |
| Escape | Reverts to last saved value; returns to display mode |

### Workspace Entry (PROJ-03)

| State | Behavior |
|-------|----------|
| Directory linked | "Open Workspace" button enabled; click opens file tree to directory + focuses terminal drawer |
| No directory | Button reads "Link Directory"; click opens OS file picker; on selection saves path and transitions to "Open Workspace" |
| No terminal session | Workspace button creates a new default terminal session via `useTerminalSessionStore.getState().createSession()` |

### Tier Badge Click (D-10)

| State | Behavior |
|-------|----------|
| Click tier badge | Opens `TierSelectionDialog` in change mode (same as current "Change plan" behavior) |

### Progress Indicator

| State | Display |
|-------|---------|
| Has tasks | `{complete}/{total}` as plain text, e.g. "3/5" |
| No tasks | Hidden (no "0/0" shown) |
