## Interaction Contract

### Toggle Buttons (D-01, D-02)

| State | Style |
|-------|-------|
| Inactive | `variant="ghost"` -- transparent background, `--color-foreground` text |
| Active | `variant="default"` -- `--color-accent` background, `--color-accent-foreground` text |
| Hover (inactive) | Standard ghost hover from shadcn theme |

Both buttons can be active simultaneously (D-05). Clicking an active button closes that panel (D-06).

### Action Buttons (D-08)

| Property | Value |
|----------|-------|
| Layout | Horizontal flex wrap, gap 8px |
| Style | shadcn `Button` with `variant="outline"`, `size="sm"` |
| Icon | Lucide icon left of label (16px) |
| Specific buttons | "Run Daily Briefing" (Sparkles icon), "Organize Calendar" (Calendar icon), "Organize Goals" (Target icon) |
| Click behavior | Phase 32: placeholder -- inserts a static message into chat output area. Real wiring deferred. |

### Jump-to-Top Button (D-08)

| Property | Value |
|----------|-------|
| Visibility | Shown only when chat output is scrolled down past the action buttons |
| Position | Fixed bottom-right of the center view, 16px from edges |
| Style | shadcn `Button`, `variant="secondary"`, `size="icon"`, circular (`rounded-full`) |
| Icon | `ChevronUp` from lucide-react, 16px |
| Accessibility | `aria-label="Back to top"` |

### Panel Dismissal (D-06)

Panels close only via toolbar toggle button. No click-outside-to-close. No escape key close.

### Panel Load State (D-10)

Both panels start closed on every hub load. No persistence.
